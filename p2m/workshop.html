🖨 Printer Fleet → Steps 1.5, 2.1, 2.3

▶
Define every printer in your workshop. Active printers determine your total production capacity. Build plate size feeds into batch printing calculations in Step 2.1.
Printer Name Model Build Plate (mm) Max Speed (mm/s) Hrs / Day Status
Feeds into: Step 1.5 uses total active printers and daily print hours to check production feasibility. Step 2.1 uses build plate dimensions for batch layout. Step 2.3 uses printer model for slicer profile selection.

💰 Machine Rates → Step 3.1 (COGS)

▶
Calculate the true cost of running each printer per hour. The blended rate across all active machines is the single number used in COGS calculations.
Cost Component Value Calculation / Notes
Key output: The Blended Hourly Rate is the single number that Step 3.1 (COGS) uses for the machine time cost component. It accounts for depreciation, electricity, and maintenance across your entire fleet.

🧵 Materials Inventory → Steps 2.3, 3.1

▶
Track your filament and consumable costs. The cost per gram is what drives the material component of COGS in Step 3.1.
Material Type Color Cost / kg Cost / gram Stock (kg) Supplier
Feeds into: Step 2.3 selects the material for the product. Step 3.1 uses the cost/gram to calculate material cost per unit (material weight from slicer x cost/gram).
